# Knowledge

Persistent knowledge accumulated by the agent across sessions.

## Status: Stub (v1)

The knowledge framework is planned but not yet implemented. Directory structure is scaffolded for future use.

## Planned Structure

```
knowledge/
├── patterns/       # Recurring patterns identified across work
├── facts/          # Domain facts, reference data, constants
├── procedures/     # Step-by-step workflows and runbooks
└── lessons/        # Lessons extracted from reflections
```

## Future Implementation

In v2, this directory will support:
- Automatic knowledge extraction from interactions via the ingesting-knowledge skill
- Cross-session pattern detection
- Searchable knowledge base for context injection

For now, agents may write free-form notes here as needed. No strict schema enforced in v1.
