# Compounding Knowledge

After completing non-trivial work, distill the episode into the agent brain's Inbox.

## When to Capture

Capture an episode when ANY of these are true:
- Something unexpected happened (good or bad)
- You made a mistake or found a better approach
- You discovered a gap in your knowledge
- A new pattern or technique emerged
- The task required non-obvious problem-solving

Do NOT capture routine, successful work where nothing was learned.

## How to Capture

Write a lightweight episode note to `02_Inbox/` in the agent brain vault:

1. Use `09_Templates/template-episode.md` as the format
2. Name the file `YYYY-MM-DD-<brief-summary>.md`
3. Keep it short — this is raw signal, not analysis
4. Tag it `type/episode`, `memory/episodic`, `workflow/draft`
5. Commit with message: `capture: <brief summary>`

## What NOT to Do

- Don't analyze or reflect yet — that's the ingestion phase
- Don't write directly to `03_Memory/` or `06_Knowledge/` — those are curated destinations
- Don't skip capture because the episode seems minor — small signals compound
- Don't write a transcript — distill to the essential signal
