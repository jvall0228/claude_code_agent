# Native app UI

## Problem Statement

Managing agents through CLI and tmux works for power users but isn't accessible. A native desktop app would provide the Slack-like thread experience with proper UI for managing agents, viewing conversations, and monitoring status.

## Acceptance Criteria

- [ ] Desktop app for agent management
- [ ] Thread view (Slack-like conversation UI)
- [ ] Agent status dashboard
- [ ] Quick actions (restart, clear, heartbeat)
